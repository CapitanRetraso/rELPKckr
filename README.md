# rELPKckr
An **ELPK** archive extractor/packer used in games by ***syn Sophia***.
<br />
<br />

### Usage
```cmd
rELPKckr <file or directory>
```
Supports drag & drop
<br />
<br />

### Disclaimer
Only tested with ***Kurohyou 2: Ryu ga Gotoku Ashura Hen***
